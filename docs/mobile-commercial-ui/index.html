Commercial Mobile UI

Nexus Platform 移动端商业化 UI 原型

这份页面集将 Android 和 iOS 统一成一套更成熟的游戏平台体验，强调游戏发现、会员转化、资产沉淀与内容运营。 页面以 HTML 平铺展示，适合先做产品讨论、视觉对齐，再拆分到原生实现。

品牌方向: 次世代游戏娱乐平台 平台策略: Android / iOS 共用视觉语言 页面覆盖: 启动、3 张引导、邮箱登录、注册、忘记密码、首页、详情、会员、运行态

核心页面总览

建议先以这套 UI 原型作为移动端产品蓝本，再逐页映射到 Android XML / Compose 与 SwiftUI。
